# Recommended Image Dimensions

Use these dimensions for consistent UI patterns.

| Use Case | Dimensions | Notes |
|----------|-----------|-------|
| **Hero (Full)** | 1920x1080 | Standard 16:9 cinematic |
| **Hero (Split)** | 1200x600 | 2:1 aspect ratio |
| **Product Card** | 800x800 | Square, clean background |
| **Feature Card** | 600x400 | 3:2 aspect ratio |
| **Profile Photo** | 400x400 | High-res avatar |
| **Social (OG)** | 1200x630 | Optimized for Facebook/LinkedIn |
| **Social (X)** | 1200x675 | Optimized for Twitter |
| **YouTube** | 1280x720 | 720p 16:9 |
| **Banner (Wide)**| 1920x400 | Very wide strip |
| **Logo/Icon** | 200x200 | Use .png for transparency |

## Format Selection
- **.png**: Best for logos, icons, illustrations, and images requiring transparency.
- **.jpg**: Best for photorealistic images and complex backgrounds to minimize file size.
